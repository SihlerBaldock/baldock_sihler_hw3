# MMED 1054 HW3 - CSS Cascade, Selection & Pseudo Selectors
This repository is for the third homework assignment in MMED 1054, and focuses on working with the CSS Cascade, Selection, and Pseudo Selectors to create a to do list with HTML and CSS.

## Installation
No installation required. Simply download the zip file and extract.

## Usage
Open index.html in the browser of your choice.
